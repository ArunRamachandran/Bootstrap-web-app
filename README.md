# Installtion process

- clone the project 

```   
git@github.com:ArunRamachandran/ProductHub-React-Redux-BoilerPlate.git
```

- Install the project dependencies

``` 
npm install
```
 
- Run in development mode

```
npm run start
```

- To run unit test cases

```
npm run test
```
 
# Production build
 
```
npm run build
```
